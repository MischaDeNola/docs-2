Stitch offers these methods for creating a replication schedule: - **Replication Frequency** - Replication jobs begin at defined intervals. [See the Replication Frequency documentation]({{ link.replication.rep-frequency | prepend: site.baseurl }}). - **Anchor Scheduling** - Replication jobs begin at defined intervals and are "anchored" to a time you select. [See the Anchor Scheduling documentation]({{ link.replication.anchor-scheduling | prepend: site.baseurl }}). {% if integration.anchor-scheduling == false %}To use Anchor Scheduling with this integration, contact Stitch Support.{% endif %} {% if integration.cron-scheduling == true %} - **Advanced Scheduling using cron expressions**. [See the Advanced Scheduling documentation]({{ link.replication.cron-scheduling | prepend: site.baseurl }}). **Note**: This feature is only available on Enterprise plans. {% endif %} To help prevent overages, consider setting the integration to replicate less frequently.
